KS2 BACKEND

## Description

ks2 - user management and property sales api

## Project setup

```bash
npm install
```

Clone the ```.env.template``` file and rename it to ```.env```

Configure database and update .env

Configure database with docker (optional)
```bash
docker compose up -d
```

## Compile and run the project

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```
